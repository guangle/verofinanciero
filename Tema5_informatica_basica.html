Tema 5: Informática Básica, Windows, Internet y Google Workspace

Informática Básica: Conceptos

La informática es la ciencia que estudia el tratamiento automático de la información mediante computadoras u ordenadores. Un sistema informático consta fundamentalmente de dos partes: Hardware y Software.

  • Hardware: Parte física del sistema informático. Son los componentes tangibles. Se divide en:
    • Unidad Central de Proceso (CPU o Procesador): El "cerebro" del ordenador, ejecuta las instrucciones de los programas.
    • Memoria Principal (RAM - Random Access Memory): Memoria volátil donde se cargan los programas y datos que se están usando en el momento. Rápida pero pierde contenido al apagar el equipo.
    • Memoria Secundaria (Almacenamiento): Dispositivos que almacenan información de forma permanente (no volátil): Discos Duros (HDD), Unidades de Estado Sólido (SSD), memorias USB, CD/DVD...
    • Periféricos: Dispositivos que conectan el ordenador con el exterior.
      • De Entrada: Permiten introducir información (Teclado, Ratón, Escáner, Micrófono).
      • De Salida: Muestran información procesada (Monitor/Pantalla, Impresora, Altavoces).
      • De Entrada/Salida (Mixtos): Realizan ambas funciones (Pantalla táctil, Multifunción -impresora/escáner-, Módem/Router).
  • Software: Parte lógica del sistema informático. Conjunto de programas, instrucciones y datos que hacen funcionar el hardware. Se divide en:
    • Software de Sistema (Sistema Operativo - SO): Programa fundamental que gestiona los recursos del hardware y sirve de interfaz entre el usuario y la máquina (Ej: Windows, macOS, Linux, Android, iOS).
    • Software de Aplicación: Programas diseñados para realizar tareas específicas para el usuario (Ej: Procesador de textos -Word, Google Docs-, Hoja de cálculo -Excel, Google Sheets-, Navegador web -Chrome, Edge-, Correo electrónico -Outlook, Gmail-, etc.).
    • Software de Programación/Desarrollo: Herramientas para crear otros programas (Compiladores, editores de código...).
  • Unidades de medida de la información:
    • Bit: Unidad mínima de información (0 o 1).
    • Byte (B): Conjunto de 8 bits. Suele representar un carácter (letra, número...).
    • Kilobyte (KB): 1024 Bytes (aprox. mil).
    • Megabyte (MB): 1024 KB (aprox. un millón).
    • Gigabyte (GB): 1024 MB (aprox. mil millones).
    • Terabyte (TB): 1024 GB (aprox. un billón).
  • Redes Informáticas: Conjunto de ordenadores y dispositivos conectados entre sí para compartir recursos e información.
    • LAN (Local Area Network): Red de área local (edificio, oficina).
    • WAN (Wide Area Network): Red de área extensa (país, continente). Internet es la WAN más grande.
    • Topologías: Forma en que se conectan los equipos (bus, estrella, anillo...).
    • Componentes: Tarjeta de red, cableado, switches, routers...

Introducción al Sistema Operativo Windows 10/11

Windows es el sistema operativo de Microsoft, uno de los más utilizados en ordenadores personales. Proporciona una interfaz gráfica de usuario (GUI) para interactuar con el ordenador.

Componentes principales de la interfaz:

  • Escritorio: Área principal de trabajo donde se muestran iconos de acceso directo, archivos y carpetas. Fondo personalizable (papel tapiz).
  • Barra de Tareas: Banda horizontal (normalmente inferior) que contiene el Menú Inicio, iconos de acceso rápido, programas abiertos, área de notificación (reloj, iconos sistema) y botón de vista de tareas/escritorios virtuales.
  • Menú Inicio: Botón (generalmente esquina inferior izquierda) que despliega un menú con acceso a aplicaciones instaladas, configuración, opciones de energía (apagar, reiniciar, suspender) y cuenta de usuario.
  • Explorador de Archivos: Herramienta fundamental para gestionar archivos y carpetas. Permite ver el contenido de las unidades de almacenamiento (Disco C:, USB...), crear, copiar, mover, renombrar y eliminar archivos/carpetas, buscar elementos, etc. Vistas personalizables (iconos, lista, detalles...). Acceso rápido a carpetas comunes (Documentos, Descargas, Imágenes...).
  • Panel de Control / Configuración: Aplicaciones que permiten ajustar las opciones del sistema, hardware, software, redes, cuentas de usuario, privacidad, etc. (En Windows 10/11, la app Configuración está reemplazando gradualmente al Panel de Control clásico).

Operaciones básicas con archivos y carpetas:

  • Archivo: Conjunto de información almacenada con un nombre y una extensión (ej: `documento.docx`, `imagen.jpg`). La extensión indica el tipo de archivo y con qué programa se abre.
  • Carpeta (o Directorio): Contenedor para organizar archivos y otras carpetas.
  • Seleccionar: Hacer clic sobre un elemento. Ctrl+Clic para selección múltiple discontinua. Shift+Clic para selección múltiple continua. Ctrl+A para seleccionar todo.
  • Copiar (Ctrl+C): Duplica un archivo/carpeta en el portapapeles.
  • Cortar (Ctrl+X): Mueve un archivo/carpeta al portapapeles (lo elimina de origen al pegar).
  • Pegar (Ctrl+V): Inserta el contenido del portapapeles en la ubicación actual.
  • Eliminar (Tecla Supr/Del): Envía el elemento a la Papelera de Reciclaje (se puede restaurar). Shift+Supr para eliminación permanente (sin pasar por la papelera).
  • Crear carpeta: Clic derecho -> Nuevo -> Carpeta.
  • Renombrar: Seleccionar -> F2 o Clic derecho -> Cambiar nombre.

La Red Internet

Internet es una red mundial descentralizada de ordenadores interconectados que utilizan la familia de protocolos TCP/IP para comunicarse. Permite acceder a una gran cantidad de servicios e información.

Conceptos clave:

  • Navegador Web (Browser): Software de aplicación que permite visualizar páginas web (Ej: Google Chrome, Microsoft Edge, Mozilla Firefox, Safari). Interpreta el código HTML, CSS y JavaScript.
  • Buscador Web (Search Engine): Sitio web que permite buscar información en Internet introduciendo palabras clave (Ej: Google, Bing, DuckDuckGo). Utilizan "arañas" o "robots" para indexar el contenido de la web.
  • World Wide Web (WWW): Sistema de distribución de documentos de hipertexto (páginas web) interconectados y accesibles a través de Internet.
  • Página Web: Documento electrónico (normalmente en HTML) que contiene texto, imágenes, enlaces, etc., accesible a través de un navegador.
  • Sitio Web: Conjunto de páginas web relacionadas y accesibles bajo un mismo nombre de dominio.
  • URL (Uniform Resource Locator): Dirección única que identifica un recurso en Internet (ej: `https://www.google.com`). Partes:
    • Protocolo: `https://` (Protocolo seguro de transferencia de hipertexto). `http://` (no seguro). Otros: `ftp://`, `mailto:`.
    • Dominio: `www.google.com` (Nombre que identifica al sitio web). Incluye subdominio (`www`), dominio de segundo nivel (`google`) y dominio de nivel superior o TLD (`.com`, `.es`, `.org`, `.gob`...).
    • Ruta: `/carpeta/archivo.html` (Opcional, indica la ubicación específica del recurso dentro del servidor).
  • Dirección IP: Número único que identifica a un dispositivo conectado a una red que utiliza el protocolo IP (ej: `192.168.1.1`, `2001:db8::1`). El DNS (Domain Name System) traduce los nombres de dominio (fáciles de recordar) a direcciones IP (que entienden las máquinas).
  • Hipervínculo (Enlace, Link): Elemento en una página web que al hacer clic te lleva a otra página o recurso.
  • Correo Electrónico (Email): Servicio para enviar y recibir mensajes electrónicos. (Ver sección específica).
  • Otros servicios: FTP (transferencia archivos), VoIP (llamadas voz), Streaming (video/audio), Redes Sociales, Mensajería Instantánea...
  • Riesgos y Seguridad:
    • Malware: Software malicioso (Virus -se replica-, Gusanos -se propagan por red-, Troyanos -parecen legítimos-, Ransomware -secuestra datos-, Spyware -espía-...). Protección: Antivirus actualizado, no descargar de fuentes dudosas.
    • Phishing: Suplantación de identidad (ej: email falso del banco) para robar credenciales o datos personales. Verificar remitente, no hacer clic en enlaces sospechosos.
    • Seguridad contraseñas: Usar contraseñas largas, complejas (mayúsculas, minúsculas, números, símbolos) y únicas para cada servicio. Considerar gestores de contraseñas y autenticación de dos factores (2FA).
    • HTTPS: Indica conexión segura (cifrada) con el sitio web. Verificar el candado en la barra de direcciones.

Trabajo Colaborativo con Herramientas de Google (Google Workspace)

Google Workspace (anteriormente G Suite) es una suite de herramientas ofimáticas, de colaboración y comunicación basada en la nube ofrecida por Google. Permite trabajar de forma conjunta y acceder a la información desde cualquier lugar con conexión a Internet.

Componentes principales (versiones gratuitas y de Workspace):

  • Google Drive: Servicio de almacenamiento en la nube. Permite guardar archivos de todo tipo y sincronizarlos entre dispositivos. Facilita la organización en carpetas y la búsqueda. Es la base para las demás herramientas.
  • Google Documentos (Docs): Procesador de textos online. Similar a Microsoft Word. Permite crear y editar documentos de texto.
  • Google Hojas de Cálculo (Sheets): Hoja de cálculo online. Similar a Microsoft Excel. Permite trabajar con datos en tablas, realizar cálculos, crear gráficos.
  • Google Presentaciones (Slides): Herramienta para crear presentaciones online. Similar a Microsoft PowerPoint.
  • Google Formularios (Forms): Permite crear encuestas y formularios online para recopilar información fácilmente.
  • Google Meet: Servicio de videoconferencias.
  • Google Chat: Servicio de mensajería instantánea para equipos.
  • Gmail: Servicio de correo electrónico. (Ver sección específica).
  • Google Calendar: Calendario online para gestionar eventos y citas.

Características clave del trabajo colaborativo:

  • Compartir: Se pueden compartir archivos y carpetas con otros usuarios, otorgando diferentes niveles de permiso (Lector, Comentador, Editor). Se comparte mediante enlace o directamente a correos electrónicos.
  • Edición simultánea (en tiempo real): Varios usuarios pueden trabajar en el mismo documento, hoja de cálculo o presentación al mismo tiempo. Se ven los cursores y cambios de los demás colaboradores.
  • Historial de versiones: Se guarda automáticamente un historial de los cambios realizados, permitiendo ver versiones anteriores y restaurarlas si es necesario.
  • Comentarios y sugerencias: Se pueden añadir comentarios en partes específicas del documento para discutir o hacer anotaciones. El modo "Sugerencias" permite proponer cambios que el propietario puede aceptar o rechazar.
  • Acceso desde cualquier lugar: Al estar basado en la nube, se puede acceder y trabajar desde cualquier dispositivo con navegador web y conexión a Internet.

Google Workspace (versiones de pago): Ofrece funcionalidades adicionales orientadas a empresas y organizaciones, como: dominios de correo personalizados (@tuempresa.com), mayor espacio de almacenamiento, controles de seguridad y administración avanzados, soporte técnico, etc.

El Correo Electrónico de Google (Gmail): Conceptos y Funcionamiento

Gmail es el servicio de correo electrónico gratuito (y parte de Workspace) de Google. Es uno de los más populares y se accede principalmente a través de interfaz web o aplicaciones móviles.

Conceptos elementales y funcionamiento:

  • Dirección de correo: Identificador único con formato `nombredeusuario@dominio.com` (ej: `usuario@gmail.com`).
  • Interfaz principal:
    • Recibidos (Inbox): Carpeta principal donde llegan los nuevos correos. Puede tener pestañas (Principal, Social, Promociones...).
    • Enviados: Mensajes que has enviado.
    • Borradores: Mensajes que has empezado a escribir pero no has enviado.
    • Spam (Correo no deseado): Carpeta donde Gmail filtra automáticamente correos sospechosos o publicitarios masivos. ¡Revisar ocasionalmente!
    • Papelera: Mensajes eliminados (se borran definitivamente tras un tiempo, 30 días).
    • Etiquetas (Labels): Sistema flexible para organizar correos (similar a carpetas, pero un correo puede tener varias etiquetas). Se pueden crear etiquetas personalizadas y asignarles colores.
    • Barra de búsqueda: Potente buscador para encontrar correos por remitente, asunto, contenido, fecha, etc. Admite operadores de búsqueda avanzada.
  • Redactar un correo nuevo:
    • Para (To): Destinatario(s) principal(es).
    • CC (Copia de Carbón): Enviar una copia a otros destinatarios (todos ven quién está en Para y CC).
    • CCO (Copia de Carbón Oculta) / BCC (Blind Carbon Copy): Enviar una copia oculta a otros destinatarios (los de Para y CC no ven a los de CCO; los de CCO no se ven entre sí). Útil para enviar a muchas personas sin revelar sus direcciones o para copias informativas discretas.
    • Asunto (Subject): Título breve y descriptivo del correo. ¡Importante ponerlo siempre!
    • Cuerpo del mensaje: Contenido principal del correo. Se puede formatear (negrita, cursiva, listas...).
    • Adjuntar archivos (Attach): Icono de clip para añadir documentos, imágenes, etc., desde el ordenador o Google Drive. Existen límites de tamaño (aprox. 25 MB por correo, si es más grande se comparte enlace de Drive).
    • Firma: Texto que se añade automáticamente al final de los correos (nombre, cargo, contacto...). Configurable.
  • Acciones con correos recibidos: Responder, Responder a todos, Reenviar, Archivar (quita de Recibidos sin borrar), Marcar como leído/no leído, Marcar como importante, Mover a (etiqueta), Filtrar mensajes similares, Bloquear remitente, Marcar como spam.
  • Filtros: Reglas automáticas para organizar correos entrantes (ej: aplicar etiqueta, archivar, eliminar... según remitente, asunto, etc.).
  • Contactos: Agenda para guardar direcciones de correo y otra información. Integrada con Google Contacts.
  • Seguridad: Además de las generales de Internet, Gmail tiene filtros de spam/phishing, advertencias de seguridad, opción de deshacer envío (unos segundos), verificación en dos pasos (2FA).

Trucos Nemotécnicos

  • Hardware vs. Software: Hardware es lo "duro", lo que puedes tocar. Software es lo "blando", lo lógico (programas).
  • RAM vs. Almacenamiento: RAM es como la mesa de trabajo (rápida, temporal, se limpia al irte). Almacenamiento (disco duro, SSD) es como el archivador (lento, permanente, guarda todo).
  • Periféricos Entrada/Salida: Piensa en la dirección de la información: ¿Entra al ordenador o Sale del ordenador?
    • Teclado/Ratón -> Entra info.
    • Monitor/Impresora -> Sale info.
    • Pantalla Táctil -> Entra (tocas) y Sale (ves).
  • Unidades de Medida (Orden): Bit -> Byte -> Kilo -> Mega -> Giga -> Tera. (Alfabeto B-K-M-G-T). Recuerda que van de 1024 en 1024 (no 1000).
  • Windows Atajos Comunes:
    • Ctrl+C / Ctrl+X / Ctrl+V: Copiar / Cortar / Pegar (¡Fundamentales!).
    • Ctrl+Z: Deshacer (¡Salvavidas!).
    • Alt+Tab: Cambiar entre ventanas abiertas.
    • Tecla Windows + E: Abrir Explorador de archivos.
    • Tecla Windows + L: BLoquear sesión.
  • HTTP vs. HTTPS: La S final es de Seguro (conexión cifrada). ¡Busca siempre la S y el candado!
  • Gmail CC vs. CCO (BCC):
    • CC: Copia Con vista (todos ven).
    • CCO/BCC: Copia Con Ocultación / Blind (Ciega) Carbon Copy (nadie ve a los de CCO).
  • Google Workspace Colaboración: Piensa en "Compartir, Comentar, Co-editar".
